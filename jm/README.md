# jm-photo

> 一个针对特定漫画网站的图片分割与重组网页工具，用于解决图片置乱显示问题。

## 项目简介

本工具专为处理 `https://jmcomicne.net/` 等网站上的漫画图片而设计。由于这些网站的原始图片在加载时会被JavaScript脚本**分割并打乱顺序**（一种反爬或图片保护机制），直接保存的图片是乱序的碎片。

`jm-photo` 通过逆向分析其前端混淆代码，实现了相同的分割逻辑，在网页端将碎片图片**重新拼接还原**为完整的漫画页面，方便用户正常阅读或保存。

### 核心功能
- **图片分割**：模拟网站脚本，按特定层数将一张完整图片分割成多个横向长条。
- **重组显示**：将分割后的碎片按正确顺序绘制在Canvas画布上，还原完整图像。
- **可配置逻辑**：核心分割参数基于漫画ID（JM号）和图片序号动态计算。

### 技术原理简述
本项目基于对网站混淆脚本（如 `jquery.photo-0.4.js`）的逆向分析。关键函数 `get_num(e, t)` 决定了分割的层数：
1.  输入参数：`e` = JM漫画ID（如 `1154471`），`t` = 图片序号（如 `00001`）。
2.  拼接字符串 `n = str(e) + str(t)`，计算其MD5值。
3.  取MD5值的**最后一个字符**，获取其ASCII码。
4.  根据JM号的数值范围（如 268850-421925 或其他），对ASCII码取模（10或8）。
5.  最终**分割层数 = (取模结果) * 2 + 2**。
6.  按此层数将图片横向切割并按正确顺序重新绘制。

### 使用步骤
1.  打开项目网页 `jm-photo`测试站 `https://jm.xiaoli.icpx.store/`）。
2. 打开`https://jmcomicne.net/`里的链接找到你想还原的漫画网页/URL有JM号
3.  在 测试站 `https://jm.xiaoli.icpx.store/`）中输入对应的JM号。
4.  选择图片CDN点击"加载图片"工具将自动计算分割参数并显示完整图片。

## ⚠  重要提醒与局限性

1.  **网站链接会变动**：目标漫画网站的域名或图片具体链接可能会更换。如果工具失效，请检查网站新的图片URL格式。
2.  **部分漫画无法分割**：由于网站脚本可能更新，或部分漫画采用了其他保护机制，**并非所有漫画图片都能保证成功分割重组**。
3.  **自行维护更新**：当工具失效时，需要您参考CSDN上相关文章的**最新分析和修改方法**，对工具中的分割逻辑（如`get_num`函数中的参数范围、模数）进行相应调整。
    - 主要参考文章：
      - [Code糕手](https://blog.csdn.net/qq_36684855/article/details/121582883)
      - [error=Nil](https://blog.csdn.net/m0_50233241/article/details/148189510)
4.  **测试站**：您可以使用 `https://jm.xiaoli.icpx.store/` 进行功能测试，但其图片链接与正式站可能不同。
